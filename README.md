# AI-Based Smart Glass for Visually Impaired People

## 📖 Overview
This project aims to develop an AI-powered smart glass that assists visually impaired people by detecting obstacles, recognizing objects, reading text aloud, and providing real-time voice guidance. The system uses Artificial Intelligence and Computer Vision to improve safety and independence.

## 🎯 Objectives
- Detect obstacles in real time.
- Recognize everyday objects using AI.
- Convert detected text into speech.
- Provide voice alerts and navigation assistance.
- Improve mobility and safety for visually impaired users.

## ✨ Features
- Real-time object detection
- Obstacle detection
- Text-to-Speech (TTS)
- Voice guidance
- Lightweight and wearable design
- AI-powered computer vision

## 🛠️ Technologies Used
- Python
- OpenCV
- YOLO (Object Detection)
- TensorFlow / PyTorch
- Raspberry Pi
- Camera Module
- Ultrasonic Sensor
- Text-to-Speech (TTS)
